云空调，便携小空调，为你的夏日带去清凉！



该项目创意来自 云游君, 使用 VUE3 + VITE + TS 构建, 未使用任何 UI 框架，无多余样式资源，打包后仅 84kb。

Features

优势

  • 🕐 随时随地打开空调
  • 📱 便携
  • 🔋 低功耗（使用 HTML CSS 而非 Canvas 绘制）
  • 🔊 静音
  • 🎮 操作简单
  • 🔧 安装便捷

劣势
